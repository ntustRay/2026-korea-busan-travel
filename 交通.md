# 釜山交通詳細筆記

> 旅行原則：大眾交通優先、少轉乘、少走路；下雨、炎熱或手上有戰利品時，最後一小段直接搭 Taxi。
>
> 路線確認日期：2026/07/23。公車班次與月台可能臨時調整，當天以 Naver Map／KakaoMap 顯示為準。

## 先準備

- 導航優先使用 **Naver Map（네이버 지도）** 或 **KakaoMap（카카오맵）**。
- 地鐵、輕軌、公車使用交通卡；Mobile Visit Busan Pass 不能代替交通卡。
- 地鐵方向看「終點站名稱」，不要只看月台左右。
- 行李多、下大雨或轉乘已經很累時，最後一段搭 Taxi，不為省小錢破壞行程。

# 7/31｜金海機場 → 飯店

## 預設：全軌道交通

預估約 70～90 分鐘，轉乘 2 次。優點是不受塞車影響。

```text
김해국제공항 국제선청사（金海機場國際航廈）
→ 跟著「경전철 / Light Rail / 공항역」指標步行
→ 공항역（機場站）搭 부산김해경전철
→ 往 사상 方向
→ 사상역（沙上站）下車
→ 轉 부산도시철도 2호선（Metro Line 2）
→ 往 장산 方向
→ 서면역（西面站）下車
→ 轉 부산도시철도 1호선（Metro Line 1）
→ 往 다대포해수욕장 方向
→ 부산역（釜山站）下車
→ 6번 출구（6 號出口）
→ 步行約 5 分鐘到 하운드호텔 부산역점
```

### 不要搭錯方向

- 機場站 → 沙上：看 **사상**。
- 沙上 → 西面：Line 2 看 **장산**。
- 西面 → 釜山站：Line 1 看 **다대포해수욕장**。

### 入境太累的替代方案

直接向 Taxi 司機出示：

```text
하운드호텔 부산역점으로 가 주세요.
부산광역시 동구 중앙대로236번길 9
```

中文意思：「請載我去 Hound Hotel Busan Station。」

# 8/01｜飯店 → Skyline Luge → Outlet → 飯店

## 飯店 → Skyline Luge Busan

預估約 60～75 分鐘軌道交通，再搭一小段 Taxi。

```text
飯店步行到 부산역 6번 출구
→ 부산역 搭 Metro Line 1
→ 往 노포 方向
→ 부전역（釜田站）下車
→ 跟著「동해선 부전역」指標步行轉乘
→ 동해선 부전역 搭 Donghae Line
→ 搭往 태화강／일광 方向，確認列車停靠 오시리아
→ 오시리아역（奧西利亞站）下車
→ 出站後搭短程 Taxi
→ 스카이라인 루지 부산
```

### 釜田轉乘注意

Metro 的 **부전역** 與 Donghae Line 的 **동해선 부전역**不是同一月台，需跟著指標步行轉乘。不要看到「부전」就直接在原站等東海線。

### 給 Taxi 司機看

```text
스카이라인 루지 부산으로 가 주세요.
부산광역시 기장군 기장읍 기장해안로 205
```

- 韓文名稱：**스카이라인 루지 부산**
- Naver Map 搜尋：`스카이라인 루지 부산`

## Skyline Luge → Lotte Premium Outlets Dongbusan

兩地同在 Osiria 區域，預設搭短程 Taxi，避免 8 月高溫下走路。

```text
롯데프리미엄아울렛 동부산점으로 가 주세요.
부산광역시 기장군 기장읍 기장해안로 147
```

- 韓文名稱：**롯데프리미엄아울렛 동부산점**
- Naver Map 搜尋：`롯데프리미엄아울렛 동부산점`

## Outlet → 飯店

購物後手上有袋子，建議由 Outlet 叫 Taxi 到 **오시리아역**；不拿重物時才考慮步行。

```text
롯데프리미엄아울렛 동부산점
→ Taxi 到 오시리아역
→ 동해선，往 부전 方向
→ 부전역下車
→ 步行轉 Metro Line 1 부전역
→ 往 다대포해수욕장 方向
→ 부산역下車
→ 6번 출구步行回飯店
```

# 8/02｜飯店 → SEA LIFE 水族館 → 海雲台 → SPA LAND → 飯店

## 飯店 → SEA LIFE Busan Aquarium

### 預設：地鐵

預估約 50～60 分鐘。

```text
부산역 搭 Metro Line 1
→ 往 노포 方向
→ 서면역下車
→ 轉 Metro Line 2
→ 往 장산 方向
→ 해운대역（海雲台站）下車
→ 3번 출구 或 5번 출구
→ 步行約 10 分鐘
→ 씨라이프 부산 아쿠아리움
```

- 韓文名稱：**씨라이프 부산 아쿠아리움**
- 韓文地址：**부산광역시 해운대구 해운대해변로 266**
- Naver Map 搜尋：`씨라이프 부산 아쿠아리움`

### 更少轉乘的公車替代

從釜山站附近搭 **1001 或 1003**，在 **해운대해수욕장**（海雲台海水浴場）站下車。優點是不用在西面轉地鐵；缺點是週末與尖峰時間可能塞車。

## 水族館 → 海雲台看海

水族館就在 **해운대해수욕장**（海雲台海水浴場）旁，離場後直接到海邊散步，不需要再搭車。

- 韓文名稱：**해운대해수욕장**
- Naver Map 搜尋：`해운대해수욕장`

## 海雲台／水族館 → SPA LAND Centum City

```text
步行回 해운대역
→ Metro Line 2
→ 往 양산 方向
→ 센텀시티역（Centum City 站）下車
→ 跟著「신세계백화점 / 스파랜드」指標
→ 12번 출구方向，可由地下連通區進入新世界百貨
→ 스파랜드（百貨 1F）
```

- 韓文名稱：**스파랜드 센텀시티**
- 所在地：**신세계백화점 센텀시티점 1층**
- 韓文地址：**부산광역시 해운대구 센텀남대로 35**
- Naver Map 搜尋：`스파랜드 센텀시티`

## SPA LAND → 飯店

```text
센텀시티역 搭 Metro Line 2
→ 往 양산 方向
→ 서면역下車
→ 轉 Metro Line 1
→ 往 다대포해수욕장 方向
→ 부산역下車
→ 6번 출구步行回飯店
```

# 8/03｜飯店 → 松島海上纜車 → 飯店 → 金海機場

## 飯店 → 松島海上纜車

### 預設：釜山站直達公車，少轉乘

公車路線可能因現場調整，出發時用 Naver Map 搜尋即時到站時間。

```text
飯店步行到 부산역 公車站
→ 搭 17、61 或 26 號公車
→ 암남동주민센터（岩南洞居民中心）下車
→ 步行約 10 分鐘
→ 송도베이스테이션
```

- 纜車韓文：**송도해상케이블카**
- 搭乘站韓文：**송도베이스테이션**
- 韓文地址：**부산광역시 서구 송도해변로 171**
- Naver Map 搜尋：`송도해상케이블카 송도베이스테이션`

### 下雨／炎熱時：地鐵＋短程 Taxi

這個版本比較穩定，也不用研究公車下車位置。

```text
부산역 搭 Metro Line 1
→ 往 다대포해수욕장 方向
→ 자갈치역（札嘎其站）下車
→ 1번 출구
→ 搭短程 Taxi 到 송도베이스테이션
```

給 Taxi 司機看：

```text
송도해상케이블카 송도베이스테이션으로 가 주세요.
부산광역시 서구 송도해변로 171
```

## 松島纜車 → 飯店取行李

最無腦版本：從 **송도베이스테이션** 直接叫 Taxi 回飯店。

想搭大眾交通：

```text
송도베이스테이션
→ 步行到 암남동주민센터
→ 搭 17、61 或 26 號公車
→ 부산역下車
→ 步行回飯店取行李
```

## 飯店 → 金海國際機場

BR163 於 19:10 起飛。**15:15 前必須離開飯店**，預留轉乘、走路與機場報到時間。

```text
飯店 → 부산역 6번 출구
→ Metro Line 1
→ 往 노포 方向
→ 서면역下車
→ 轉 Metro Line 2
→ 往 양산 方向
→ 사상역下車
→ 轉 부산김해경전철
→ 往 가야대 方向
→ 공항역下車
→ 跟著「국제선청사 / International Terminal」指標
→ 김해국제공항 국제선청사
```

### 不要搭錯方向

- 釜山站 → 西面：Line 1 看 **노포**。
- 西面 → 沙上：Line 2 看 **양산**。
- 沙上 → 機場：輕軌看 **가야대**。

# 雨備與體力不足時的交通原則

1. **Luge 日**：Osiria Station 與 Luge／Outlet 之間全部搭短程 Taxi。
2. **水族館＋SPA 日**：維持地鐵，不增加海岸列車或廣安里。
3. **松島日**：小雨可改「地鐵到札嘎其＋Taxi」；大雨或強風取消纜車。
4. **回程日**：不從松島直接去機場，先回飯店取行李，再走固定軌道路線。
5. **迷路處理**：先在 Naver Map 貼韓文名稱；仍找不到時，把韓文地址直接給站務員或 Taxi 司機看。
