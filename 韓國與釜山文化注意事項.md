# 韓國與釜山文化注意事項

> 這份筆記以「旅客實際會遇到的場合」為主。多數是禮貌習慣，不是法律；看不懂時，跟著現場標示與其他客人的做法即可。

## 30 秒快速版

| 場景 | 建議做法 | 避免 |
|---|---|---|
| 打招呼 | 說 `안녕하세요`，輕微點頭 | 把店員沒一直微笑解讀成不友善 |
| 付款／拿東西 | 用雙手或右手、左手輕扶手腕 | 隨手丟錢或把卡片甩在桌上 |
| 地鐵／公車 | 排隊、先下後上、音量放低 | 擋門、擋走道、開擴音通話 |
| 餐廳 | 看是否 Self-service，使用桌上呼叫鈴 | 主動給小費、把筷子直插飯裡 |
| 傳統市場 | 先問價格、單位與料理費 | 預設每一攤都能殺價、沒問就拍人 |
| SPA／澡堂 | 先洗澡再入池，浴場內不拍照 | 穿泳衣進大浴池、拿手機拍攝 |
| 海邊／景點 | 垃圾帶走、拍人前先詢問 | 長時間擋路、近距離拍陌生人 |

## 釜山給人的感覺

- 釜山是港口城市，海鮮市場、商港、海灘與山坡聚落文化都很明顯。
- 市場或餐廳裡可能聽到較有力、起伏較大的釜山方言（부산 사투리）；聽起來比較直接，不一定代表對方生氣。
- 札嘎其市場常見代表性釜山方言：`오이소, 보이소, 사이소`，意思接近「來看看、看一看、買一買」。
- 釜山觀光區對外國旅客相當習慣；不會韓文沒關係，先禮貌打招呼，再使用 Papago 或指圖片即可。

## 打招呼與互動

### 建議做法

- 進店或詢問時先說：`안녕하세요`（Annyeonghaseyo，你好）。
- 離開、收到協助或結帳後說：`감사합니다`（Gamsahamnida，謝謝）。
- 對店員、司機與年長者輕微點頭即可，不需要正式鞠躬。
- 付款、遞護照、拿找零或收物品時，用雙手最有禮貌；旅客只用右手通常也不會失禮。
- 不確定對方是否在排隊時，先觀察或問一下，不要直接往前站。

### 不用過度緊張

- 韓國服務人員不一定全程微笑或閒聊，語氣也可能較俐落，通常只是工作節奏，不代表不耐煩。
- 英文溝通不順時，不要連續提高音量；改用短句、圖片、地址或翻譯 App 更有效。
- 不需要刻意模仿韓劇中的正式禮節，保持禮貌、不要造成別人困擾就足夠。

## 餐廳與吃飯

### 基本習慣

- 韓國一般沒有小費文化，正常用餐不需另外留小費。
- 許多餐廳的水、小菜或餐具是 Self-service；先找 `셀프` 標示，再決定是否叫店員。
- 桌上常有呼叫鈴，可直接按鈴請店員過來；不用一直揮手或大聲喊。
- 湯、飯通常用湯匙；筷子主要夾菜，不必勉強全程只用筷子。
- 不要把筷子直立插在白飯上，這在韓國也容易讓人聯想到祭祀。
- 共食小菜很常見，但不要用自己咬過的筷子在公共盤中翻找太久。

### 烤肉店

- 有些店由店員全程烤肉；店員正在處理時，不要搶著翻肉，等對方示意即可。
- 生肉夾與吃熟食的筷子分開使用。
- 外套可放進椅子下方或店家提供的袋子，避免沾上烤肉味。
- 生菜包肉通常一口吃掉，但這是吃法，不是硬性規則。

### 與韓國人正式吃飯時

- 若同桌有明顯年長者，可等對方開始吃再動筷，會比較有禮貌。
- 倒酒或接酒時用雙手較正式；不想喝酒可以直接禮貌拒絕，不需要勉強。
- 只有兩人自己旅行用餐時，不必一直遵守正式長幼順序。

## 地鐵、公車與電扶梯

- 排隊候車，先讓車內乘客下車再上車。
- 車廂內把對話、影片與通話音量放低，手機不要開擴音。
- 優先席留給長者、孕婦、身心障礙者與有需要的人；普通座位也應主動讓給明顯有需要者。
- 行李不要擋車門、走道與緊急出口；車輛行駛時要扶好或固定。
- 搭釜山市區公車通常由前門上車、下車前按鈴，再從後門下車。
- 電扶梯依現場人流站立，不要兩人並排擋住通道；最安全仍是站穩並握扶手。

## 傳統市場與海鮮店

### 先確認再點餐

在札嘎其市場、海雲台市場等地購買海鮮或料理時，先確認：

1. 價格是每隻、每盤、每公斤，還是整份價格。
2. 是否另收切工費、料理費、座位費或樓上餐廳費用。
3. 最終總價是多少，再決定是否購買。

### 殺價與付款

- 傳統市場不代表每一攤都能殺價；近年部分市場推動固定價格與刷卡服務。
- 沒有標價時可以禮貌詢價，買較多時再問一次是否有優惠，不要強勢反覆砍價。
- 小攤可能偏好現金，但能刷卡的店家愈來愈多；付款前先確認即可。

### 拍照

- 拍商品或市場全景通常問題不大，但近距離拍店員、攤販、料理過程前先詢問。
- 不要把手機直接伸到正在切魚、煮食或收錢的人面前。
- 對方拒絕時直接收起手機，不必追問原因。

## SPA LAND／韓式澡堂

韓式澡堂通常分成「男女分開的大浴場」與「穿館內服的共用汗蒸幕區」。實際規則以 SPA LAND 現場標示為準。

### 大浴場

- 先在淋浴區把身體洗乾淨，再進公共浴池。
- 大浴場通常不穿泳衣；毛巾不要泡進浴池。
- 更衣室、淋浴區與浴池內禁止拍照，手機應收好。
- 長頭髮綁起來，不要讓頭髮泡在池水中。
- 不在池內搓洗身體、游泳、潑水或大聲聊天。

### 共用汗蒸幕區

- 穿場館提供的衣服進入共用區域。
- 館內手環或鑰匙要保管好，部分消費會記在手環上，離場時一起結帳。
- 第一次使用高溫房先短時間體驗，補充水分；不舒服就立刻離開。
- SPA LAND 基本入場時間有上限，超時可能加收費用，進場時確認截止時間。

## 鞋子、室內空間與垃圾

- 一般飯店、百貨公司與現代餐廳不用脫鞋。
- 傳統餐廳、民宿、寺廟或有木地板的空間，看到鞋櫃、拖鞋或其他人脫鞋時再跟著做。
- 不要穿著室外鞋踩上需脫鞋的座位區或地板區。
- 公共垃圾桶可能不像台灣便利商店周邊那麼容易找到；隨身準備小袋子，找到正確垃圾桶後再丟。
- 店內飲食產生的垃圾，依店員指示放在回收區或留在桌面，不要自行塞入不明桶子。

## 海灘、景點與拍照

- 不亂丟垃圾、不在公共步道長時間擺腳架或行李擋路。
- 拍攝陌生人、孩童、泳客或情侶前先詢問，不要持續近距離跟拍。
- 寺廟與紀念設施降低音量，不跨越圍欄、不碰供品或宗教物件。
- 景點標示禁止 Selfie stick、腳架、飲食或攝影時，以現場規則為準。
- 海邊飲酒、吸菸、煙火與夜間活動限制可能依海灘及時段不同，先看現場告示，不要預設一定可以。

## 最實用韓文

| 韓文 | 中文 | 使用時機 |
|---|---|---|
| 안녕하세요 | 你好 | 進店、詢問 |
| 감사합니다 | 謝謝 | 結帳、接受協助 |
| 죄송합니다 | 不好意思／抱歉 | 擋到人、需要打擾 |
| 괜찮아요 | 沒關係／不用了 | 婉拒或表示沒事 |
| 이거 주세요 | 請給我這個 | 指著商品點餐 |
| 얼마예요? | 多少錢？ | 市場、商店 |
| 안 매운 거 있어요? | 有不辣的嗎？ | 點餐 |
| 사진 찍어도 돼요? | 可以拍照嗎？ | 市場、店內 |
| 화장실 어디예요? | 廁所在哪裡？ | 店家、景點 |
| 여기요 | 不好意思／這裡 | 餐廳呼叫店員 |
