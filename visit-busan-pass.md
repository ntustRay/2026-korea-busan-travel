# Visit Busan Pass 購買與使用筆記

## 結論

這次建議購買 **VISIT BUSAN PASS BIG 5（Mobile）**，不要買 24H／48H，也不要買實體卡。

- **價格**：₩65,000／人
- **兩人費用**：₩130,000
- **適合原因**：不受 24／48 小時倒數限制，戶外行程可依天氣互換
- **預估節省**：以 Skyline Luge＋SPA LAND＋松島透明纜車計算，每人約省 ₩19,000，兩人約省 ₩38,000（約 NT$800 左右，依匯率變動）

## 可以現買現用嗎？

**可以。Mobile Pass 購買完成後，可以直接在 VISIT BUSAN PASS App／官網帳號的 `MY PASS` 顯示 QR Code 使用，不需要兌換實體卡。**

但不建議站在景點售票口才開始註冊與付款，避免：

- 海外信用卡付款失敗
- 手機網路不穩
- 忘記帳號密碼
- App 登入或 QR Code 顯示問題

### 建議購買時間

- **7/30 晚上**：確認釜山天氣、Luge 與纜車營運後購買
- 最晚可在第一個景點出發前購買

## 這次的 BIG 5 使用方式

BIG 5 可使用：

- A 組 2 個景點
- B 組 3 個景點
- 每個景點只能使用一次

### A 組：確定使用

1. **Skyline Luge Busan**
2. **SPA LAND Centum City**

### B 組：主要與備案

1. **松島海上纜車**：主要行程，包含 Crystal Cruise 透明車廂來回
2. **釜山塔**：松島纜車因大雨或強風停駛時的近距離備案
3. **海雲台海岸列車**：當天有體力、天氣合適才使用，不為回本硬塞

## 水族館與 X the SKY

### SEA LIFE Busan Aquarium

- 水族館是這次海雲台的**優先行程**
- BIG 5 不包含免費入場，只提供現場成人票 **30% 折扣**
- 現場成人原價 ₩32,000，折後約 ₩22,400
- 購票前仍要比較 SEA LIFE 官網預售價；若低於 ₩22,400，直接買水族館官網票較便宜

### BUSAN X the SKY

- X the SKY 是水族館的天候／臨時備案
- X the SKY 屬於 BIG 5 的 A 組
- 目前 A 組已預留給 Luge 與 SPA LAND，因此只有在 **Luge 或 SPA 其中一項取消且尚未使用兩個 A 組名額**時，才改用 X the SKY

## 使用注意事項

1. 必須開啟 `MY PASS` 的即時 QR Code；**截圖或列印 QR Code 無效**。
2. 第一次使用免費景點後，BIG 5 會啟用，之後可在 180 天內使用剩餘名額。
3. Pass 自購買日起最長有效期為 1 年；不影響本次 4 天旅程。
4. Mobile Pass **不能當交通卡**，地鐵、公車、輕軌仍需另外使用交通卡或購票。
5. 景點使用後無法退款；購買前先確認天氣與主要設施是否營運。
6. 2026/02/10 起實體 Card Pass 暫停販售，這次直接購買 Mobile Pass。
