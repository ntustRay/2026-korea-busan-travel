# 釜山 4 天 3 夜｜正常行程

> 旅遊風格：無腦放鬆、大眾交通優先、每天少量重點、不跨區折返。
>
> 住宿：Hound Hotel Busan Station（釜山站／草梁站步行約 5 分鐘）

## 行程設計

優先順序：

1. Skyline Luge Busan 滑坡車
2. 松島海上纜車
3. 購物
4. SPA LAND Centum City
5. SEA LIFE Busan Aquarium
6. 至少一天看海

這次不把 Lotte World Adventure Busan 硬塞進行程。它和 Luge、Outlet 同區，但三個一起排會太趕；預設以水族館取代樂園，較符合放鬆與雨天備案。

## 行程總覽

| 日期 | 區域 | 主要安排 |
|---|---|---|
| 7/31（五） | 釜山站 | 抵達、Check-in、附近吃飯、飯店休息 |
| 8/01（六） | 東釜山 Osiria | Skyline Luge → Lotte Premium Outlets Dongbusan |
| 8/02（日） | 海雲台／Centum City | 水族館 → 海雲台看海 → SPA LAND |
| 8/03（一） | 松島／南浦 | 松島海上纜車 → 午餐 → 機場 |

## Day 1｜7/31（五）抵達釜山

### 路線

```text
金海機場國際航廈
→ Airport Station 搭 Busan-Gimhae Light Rail
→ Sasang 沙上站轉 Metro Line 2
→ Seomyeon 西面站轉 Metro Line 1
→ Busan Station 釜山站
→ 步行約 5 分鐘至飯店
```

### 時間安排

| 時間 | 安排 |
|---|---|
| 18:00 | BR164 抵達 PUS |
| 18:00～19:15 | 入境、領行李、交通卡／網路確認 |
| 19:15～20:30 | 搭大眾交通前往飯店 |
| 20:30 | Check-in |
| 21:00 | 釜山站或草梁站附近晚餐 |
| 22:00 後 | 飯店休息；有體力再使用 Rooftop Pool |

### 原則

- 抵達日不排正式景點。
- 行李多或太累時，可直接搭 Taxi 到飯店；不要為了省一段交通把第一晚弄得很累。
- 飯店可免費寄放行李，回程日會用到。

### 當日雨備

行程不變，全部以機場、軌道交通、飯店與附近餐廳為主。

---

## Day 2｜8/01（六）Skyline Luge＋東釜山 Outlet

### 為什麼排同一天

Skyline Luge 與 Lotte Premium Outlets Dongbusan 都位於 Osiria 區域。上午先玩戶外設施，下午再購物，不必帶著戰利品玩滑坡車，也不需跨區。

### 路線

```text
Busan Station 釜山站搭 Metro Line 1
→ Bujeon Station 釜田站
→ 步行轉乘 Donghae Line 東海線
→ Osiria Station 奧西利亞站
→ 短程 Taxi 到 Skyline Luge
→ 短程 Taxi 或步行前往 Outlet
```

官方建議從釜山站搭 Line 1 到釜田站，再轉 Donghae Line 至 Osiria。Osiria 站到 Outlet 約步行 17 分鐘；考量 8 月炎熱與購物行李，本站與景點間優先使用短程 Taxi。

### 時間安排

| 時間 | 安排 |
|---|---|
| 08:40 | 從飯店出發 |
| 10:00～12:15 | Skyline Luge，建議買 3 次方案 |
| 12:15～12:30 | 前往 Outlet |
| 12:30～13:30 | Outlet 午餐 |
| 13:30～18:30 | Outlet 購物、咖啡、休息 |
| 18:30～19:30 | 晚餐或最後採買 |
| 19:30 後 | Osiria → Bujeon → Busan Station 回飯店 |

### 不加的景點

- 不加海東龍宮寺。
- 不加海雲台。
- 不加 Lotte World Adventure Busan。

以上景點雖然在東側，但加入後會變成趕場，違反這趟的核心風格。

### 當日雨備

- 小雨：先看 Skyline Luge 官網即時營運狀態。官方說明小雨通常可營運，但雷雨、強風或大雨會暫停。
- 大雨／強風／雷雨：整天與 Day 3 互換，改走「水族館＋SPA LAND」。
- 不在現場硬等天氣轉好；等待超過 30 分鐘就改雨備。

---

## Day 3｜8/02（日）水族館＋海雲台看海＋SPA

### 為什麼排一起

SEA LIFE Busan Aquarium 位於海雲台，SPA LAND 位於 Centum City，兩地都在 Metro Line 2 上。當天只有兩個主要室內景點，中間加入短時間海灘散步即可。

### 路線

```text
Busan Station 搭 Metro Line 1
→ Seomyeon 西面站轉 Metro Line 2
→ Haeundae 海雲台站
→ 步行至 SEA LIFE Busan Aquarium
→ Haeundae 海雲台站搭 Line 2
→ Centum City Station
→ SPA LAND
→ Line 2 到 Seomyeon，轉 Line 1 回 Busan Station
```

### 時間安排

| 時間 | 安排 |
|---|---|
| 09:20 | 從飯店出發 |
| 10:30～12:30 | SEA LIFE Busan Aquarium |
| 12:30～14:00 | 海雲台午餐 |
| 14:00～14:45 | 海雲台海灘散步、看海 |
| 14:45～15:20 | Line 2 前往 Centum City |
| 15:30～19:00 | SPA LAND，控制在 4 小時內 |
| 19:00～20:00 | Shinsegae Centum City 晚餐 |
| 20:00 後 | 回飯店 |

### 放鬆原則

- 水族館抓約 2 小時即可，不追每一場表演。
- 海灘只散步 30～45 分鐘，不再往尾浦、青沙浦延伸。
- SPA LAND 一般入場最多 4 小時，超時會加價。
- 如果前一天 Outlet 買太累，這天可延後 30～60 分鐘出門。

### 當日雨備

- 水族館與 SPA LAND 都是主要室內行程，正常保留。
- 取消海雲台海灘散步，午餐後直接去 Centum City。
- 多出的時間留給 SPA、Shinsegae 百貨或提早回飯店，不新增其他景點。

## Day 4｜8/03（一）松島海上纜車＋回程

### 核心原則

回程航班為 19:10。這天只排靠近釜山站西側的松島，不去海雲台或 Osiria，並設定硬性回飯店與出發時間。

### 路線

```text
飯店退房並寄放行李
→ Busan Station 搭 Metro Line 1
→ Jagalchi 札嘎其站
→ 短程 Taxi 到 Songdo Bay Station
→ 松島海上纜車來回
→ Taxi 回札嘎其／南浦吃午餐
→ Line 1 回 Busan Station 取行李
→ Line 1 到 Seomyeon
→ Line 2 到 Sasang
→ Busan-Gimhae Light Rail 到 Airport Station
```

### 時間安排

| 時間 | 安排 |
|---|---|
| 08:30 | 早餐、整理行李 |
| 09:00 | 退房並寄放行李 |
| 09:15 | 前往札嘎其，再搭短程 Taxi 到松島 |
| 10:00～11:45 | 松島海上纜車來回；對岸短暫看海，不走長程步道 |
| 12:15～13:30 | 札嘎其／南浦午餐 |
| 13:30～14:15 | 回飯店取行李 |
| 15:15 前 | 必須從飯店出發前往機場 |
| 約 16:30～17:00 | 抵達金海機場國際航廈 |
| 19:10 | BR163 起飛 |

### 纜車選擇

- 購買來回票，不買單程。
- 預設普通 Air Cruise 即可；透明底 Crystal Cruise 視怕高程度與價差決定。
- 2026 年 7～8 月官方公告營運時間為 09:00～22:00，但仍可能因天候調整。

### 當日雨備

- 小雨且官網顯示正常營運：可以搭，但視野可能較差。
- 大雨、強風或停駛：取消纜車，改去 Nampo 的 Lotte Department Store Gwangbok 室內購物／午餐。
- 不把取消的纜車改塞到東釜山；回程日以準時到機場為第一優先。

---

## 訂票與確認順序

1. **Skyline Luge**：先買可改期或可退票方案；出發當天確認即時營運狀態。
2. **SEA LIFE Busan Aquarium**：可先買電子票，雨天最穩定。
3. **SPA LAND**：通常現場購票即可，預留最多 4 小時。
4. **松島海上纜車**：建議前一天或當天依天氣決定，不要太早買不可退票。
5. **Outlet**：不需預約。
