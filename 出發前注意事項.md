## 電壓、插座與充電

韓國使用 **220V、60Hz、圓形雙孔插座（Type C／F）**；台灣常見為 110V、扁形插頭，插頭形狀與電壓都不同。

### 可以直接使用

看到充電器標示以下內容，通常只需要轉接頭，不需要變壓器：

```text
Input: 100-240V, 50/60Hz
```

常見可直接使用：

- iPhone／Android 手機充電器
- MacBook／筆電充電器
- 相機與行動電源充電器
- 多數 USB-C 充電器

### 不要直接插

若電器只標示 `110V`，不可直接接韓國 220V，可能燒毀或造成危險。

- 台灣 110V 吹風機
- 110V 電棒捲／離子夾
- 110V 電熱用品

**最省事做法：不要帶高功率美髮電器，直接使用飯店吹風機。**

### 建議攜帶

- 每人 1 個 Type C／F 轉接頭。
- 需要同時充多台裝置時，帶支援 `100-240V` 的多孔 USB-C 充電器。
- 不要使用只支援 110V 的延長線或排插。

官方說明：https://english.visitkorea.or.kr/svc/contents/contentsView.do?vcontsId=140059

## eSIM 與手機設定

### 建議方案

這次只有 4 天，兩人各買一張 **韓國 5 天 Data-only eSIM** 最穩定。

- 建議：Unlimited，或每日 3GB 高速流量後降速。
- 不建議只靠一人分享 Hotspot，兩人分開行動或手機沒電時會失聯。
- 韓國門號不是必要條件；若方案附韓國電話號碼可作為加分，但 Data-only 已足夠地圖、通訊與電子票券使用。

### 購買前確認

- [ ] 手機型號支援 eSIM。
- [ ] 手機已解鎖，能使用其他電信商方案。
- [ ] 方案涵蓋韓國，效期至少 5 天。
- [ ] 了解效期是「安裝後開始」還是「連上韓國網路後開始」。
- [ ] QR Code 通常只能安裝一次，不要反覆刪除 eSIM。

### 抵達韓國後設定

```text
行動數據：韓國 eSIM
數據漫遊：韓國 eSIM 開啟
台灣 SIM 數據漫遊：關閉
語音／簡訊：保留台灣 SIM，方便接收 OTP
允許行動數據切換：關閉，避免偷跑台灣漫遊費
```

若無法上網：

1. 開關飛航模式。
2. 重新啟動手機。
3. 確認韓國 eSIM 已開啟 Data Roaming。
4. 依供應商說明檢查 APN。
5. 使用機場 Wi-Fi 聯絡 eSIM 客服。

## 行動電源與飛機行李

本次搭乘長榮，規則以長榮最新公告為準。

### 行動電源

- **每人最多 2 顆**。
- 建議只帶 **100Wh 以下**產品；常見 10,000mAh、20,000mAh 通常低於 100Wh，但仍要看機身標示。
- **只能放隨身行李，禁止托運**。
- 接點需防短路：保護蓋、絕緣膠帶或分開放入保護袋。
- 應隨身保管或放前方座位下，**不要放頭頂置物櫃**。
- 飛行途中禁止使用行動電源替裝置充電，也禁止替行動電源充電。
- 藍牙耳機充電盒含鋰電池，也應放隨身行李。
- 膨脹、破損或看不清容量標示的行動電源不要帶。

長榮官方：

- https://www.evaair.com/en-tw/fly-prepare/flying-with-eva/safety-and-health-care/inflight-safety/cabin-safety/
- https://www.evaair.com/en-au/fly-prepare/baggage/additional-baggage-information/restrictions/

### 液體、噴霧與凝膠

隨身行李：

- 每個容器容量不得超過 **100ml**，不是看剩餘量。
- 全部裝入一個不超過 **1L** 的透明可密封袋。
- 每人限一袋。

超過 100ml 的洗髮精、防曬、乳液等請放托運行李；高壓噴霧、易燃品仍可能受限制。

### 刀具與尖銳物

剪刀、刀具、工具等不要放隨身行李。小型物品是否可托運仍以航空公司與安檢判定為準。

## 韓國不要帶／需要申報的物品

### 最簡單原則

**不確定是不是食品、動植物或藥品管制物，就不要帶；已經帶了就主動申報。**

### 建議完全不要帶

- 肉類、香腸、肉乾、肉鬆、含肉製品。
- 新鮮水果、蔬菜、植物、種子、土壤。
- 不確定是否含真實肉類成分的泡麵、調理包或零食。
- 大麻、CBD、THC、Hemp 等相關食品、精油、藥品或保養品。
- 電擊器、防狼噴霧、槍械、刀劍、爆裂物。
- 來源不明或疑似仿冒商品。

韓國海關將動物、肉品、植物、水果、蔬菜、農漁畜產品、加工品與其他食品列為需要申報／檢疫項目。

官方說明：https://customs.go.kr/kcs_english/cm/cntnts/cntntsView.do?cntntsId=2743&mi=8069

## 常備藥品

### 建議攜帶

依個人平常使用經驗準備少量即可：

- [ ] 止痛／退燒藥
- [ ] 腸胃藥、胃藥
- [ ] 止瀉藥
- [ ] 暈車藥
- [ ] 過敏藥
- [ ] 感冒症狀藥
- [ ] OK 繃、消毒棉片
- [ ] 防蚊液、止癢藥膏
- [ ] 個人固定處方藥

### 攜帶方式

- 保留藥品的**原包裝、成分與藥名標示**，不要全部拆成無標示藥盒。
- 處方藥攜帶本次旅程合理用量。
- 建議準備英文處方、診斷證明或醫師證明，尤其是針劑、安眠／鎮靜、精神科、ADHD、強效止痛等藥品。
- 重要藥品放隨身行李，不要全部托運。

### 管制藥品

含 Narcotics／Psychotropic controlled substances 的自用藥，入境韓國前需要取得韓國 MFDS 許可，不分國籍。

- 官方建議審查時間約 **10 個工作天**，實務上至少提前 2～3 週確認與申請。
- 不確定成分是否受管制時，先向開藥醫師、藥師或 MFDS 確認，不要直接攜帶闖關。

MFDS 官方說明：https://www.mfds.go.kr/eng/brd/m_60/view.do?seq=75669

## 付款、交通與 App

### 付款

- 信用卡為主，但仍準備少量韓元現金。
- 建議兩人先準備約 **₩100,000～₩200,000**，用於交通卡儲值、小店與臨時支出。
- 出發前確認信用卡已開啟海外交易，並記住 PIN。
- 海外刷卡遇到幣別選擇時，通常選 **KRW 韓元**，不要選 TWD 動態換匯。

### 交通卡

- Visit Busan Pass Mobile **不能當交通卡**。
- 抵達後購買／儲值 T-money 或 Cashbee，用於地鐵、公車與輕軌。
- 交通卡儲值常需要現金，因此要保留小額韓元。

### 必裝 App

- **NAVER Map**：韓國主要導航
- **KakaoMap**：第二導航備援
- **Papago**：韓文翻譯
- **VISIT BUSAN PASS**：顯示即時 QR Code
- **EVA Air**：報到、航班通知
- **Google Maps**：搜尋與收藏可用，但韓國大眾運輸導航以 NAVER／Kakao 為主
