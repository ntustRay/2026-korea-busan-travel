## 入境文件

- 台灣護照於 **2026/12/31 前暫免 K-ETA**，本次不需另外申請。
- 沒有有效 K-ETA 時，需於抵達韓國前 3 天內填寫免費 e-Arrival Card；本次可從 **7/28 起**填寫。
- 入境時隨身保留：護照、回程機票、飯店訂房資料、旅遊保險與 e-Arrival Card 完成截圖。
- e-Arrival Card 官方申請免費；要求刷卡付款的網站不要使用。

## 電壓與充電

韓國使用 **220V、60Hz、Type C／F 圓形雙孔插座**。台灣常見為 110V 扁形插頭，因此通常需要轉接頭。

- 手機、筆電、相機與多數 USB-C 充電器，只要標示 `100-240V`，可直接搭配轉接頭使用。
- 只標示 `110V` 的吹風機、電棒捲、離子夾與電熱用品不可直接插入韓國插座。
- 最省事做法：不帶台灣高功率美髮電器，使用飯店吹風機。
- 不要攜帶只支援 110V 的延長線或排插。

兩人各自使用 eSIM，不建議全程只靠一人分享 Hotspot。若無法上網，依序檢查飛航模式、重新開機、Data Roaming 與供應商 APN 設定。

## 飛機行李

### 行動電源

- 長榮自 2026/03/31 起，**每人最多攜帶 2 顆**。
- 只能放隨身行李，禁止托運；建議攜帶有清楚容量標示且低於 100Wh 的產品。
- 接點需防短路，可使用保護蓋、絕緣膠帶或獨立收納袋。
- 應隨身保管或放前方座位下，不要放頭頂置物櫃。
- 飛行途中禁止使用行動電源充電，也不可替行動電源充電。
- 膨脹、破損或容量標示不清的行動電源不要帶。

### 液體與尖銳物

- 隨身液體、噴霧與凝膠：每個容器最多 **100ml**，全部放入一個不超過 **1L** 的透明密封袋，每人一袋。
- 超過 100ml 的洗髮精、防曬、乳液等放托運行李。
- 剪刀、刀具與工具不要放隨身行李。

## 韓國不要帶的物品

最簡單原則：**不確定是否需要檢疫或申報，就不要帶；已攜帶則主動申報。**

- 肉類、香腸、肉乾、肉鬆及其他含肉製品。
- 新鮮水果、蔬菜、植物、種子與土壤。
- 成分不明的泡麵、調理包或食品。
- 大麻、CBD、THC、Hemp 相關食品、精油、藥品或保養品。
- 電擊器、防狼噴霧、槍械、刀劍與爆裂物。

## 常備藥品

準備個人平常使用過、知道怎麼服用的少量藥品即可：

- [ ] 止痛／退燒藥
- [ ] 胃藥、腸胃藥、止瀉藥
- [ ] 暈車藥
- [ ] 過敏藥、感冒症狀藥
- [ ] OK 繃、消毒棉片、止癢藥膏
- [ ] 個人固定處方藥

攜帶原則：

- 保留原包裝、藥名與成分標示，不要全部拆成無標示藥盒。
- 處方藥只帶本次旅程合理用量，重要藥品放隨身行李。
- 安眠／鎮靜、精神科、ADHD、強效止痛、針劑等藥品，建議準備英文處方或醫師證明。
- 含韓國管制成分的自用藥需事前取得 MFDS 許可，至少提前 2～3 週確認。

## 衣物與隨身用品

7 月底至 8 月初的釜山通常炎熱、潮濕且可能下大雨。

- [ ] 輕薄快乾衣物、替換襪子
- [ ] 折疊傘或輕便雨衣
- [ ] 防滑、可淋雨的鞋
- [ ] 防曬乳、帽子、防蚊液
- [ ] 輕薄外套，應付地鐵與商場冷氣
- [ ] 防水手機袋、夾鏈袋或小垃圾袋

## 付款、交通與 App

- 信用卡為主，出發前確認海外交易功能並記住 PIN。
- 海外刷卡遇到幣別選擇時選 **KRW 韓元**，不要選 TWD 動態換匯。
- 兩人準備約 **₩100,000～₩200,000** 現金，用於交通卡儲值、小店與臨時支出。
- Visit Busan Pass Mobile 不能當交通卡；抵達後另外購買或儲值 T-money／Cashbee。
- 導航以 **NAVER Map** 為主，Papago 負責翻譯；KakaoMap 可作備援，不必同時研究太多 App。
